Conditional probability of 2D Gaussian

Question Let $X,Y$ be standard normal with $\operatorname{Cov}(X,Y)=\tfrac{1}{\sqrt{2}}$. Compute $\mathbb{P}[X>0 \mid Y<0]$. Solution We write $$ \mathbb{P}[X>0 \mid Y<0] = \frac{\mathbb{P}(X>0,\, Y<0)}{\mathbb{P}(Y<0)} = 2,\mathbb{P}(X>0,\, Y<0), $$ since $\mathbb{P}(Y<0)=\tfrac12$. Because $$ Y = \frac{1}{\sqrt{2}}\,X + \frac{1}{\sqrt{2}}\,Z, $$ with $Z$ independent of $X$, the vector $(X,Z)$ is rotationally symmetric in $\mathbb{R}^2$. The event $$ X>0,\qquad Y<0 $$ corresponds to the wedge where $X>0$ but $X+Z<0$. Expected value of exponential Wiener process

Question Let $W_t$ be the standard Wiener process, what is $\mathbb{E}[e^{W_t}]$? Solution We will use the moment generating function (MGF) for $W_t\sim\mathcal{N}(0,t)$. The MGF for Gaussian is $M(s) = \exp\left(\mu s+ (1/2)\sigma^2s^2\right)$, plug in $\mu=0, \sigma=\sqrt{t}$, we have $M(t) = e^{(1/2)s^2}$ and $M(1) = \mathbb{E}[e^{W_t}] = \exp((1/2)t)$.
